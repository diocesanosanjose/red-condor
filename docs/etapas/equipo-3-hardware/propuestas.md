Propuestas para Red Cóndor - Hardware
Botón de Pánico Robusto: Proponer el uso de un botón industrial de alto relieve o un interruptor de seguridad protegido para evitar falsas alarmas, pero que sea operable con las manos entumecidas por el frío
.
Mantenimiento Modular: Diseñar el interior de las carcasas de forma que, si una batería o una antena se rompe, el propio equipo de la escuela pueda reemplazarla rápidamente sin tener que tirar todo el nodo
.
Test de Estanqueidad Escolar: Crear un protocolo de pruebas en el laboratorio del colegio sumergiendo las cajas en agua helada para validar que no existan filtraciones antes de entregarlas a los puesteros
.
Indicadores LED de Estado: Incluir luces LED simples (verde para "conectado", rojo para "batería baja") para que el puestero sepa si su equipo está funcionando correctamente sin necesidad de mirar una pantalla compleja
.
