# Equipo 4: Usuarios

**Objetivo:** Comprender quiénes utilizarían el sistema y qué problemas concretos necesitan resolver en su contexto real.

## Temas a investigar

| Tema | Qué hay que responder |
|------|----------------------|
| **Puesteros** | ¿Quiénes son los puesteros de Malargüe? ¿Dónde viven? ¿Cuál es su ciclo anual de trabajo (veranada/invernada)? ¿Cuántos días pasan en la montaña sin bajar? ¿Qué herramientas de comunicación usan hoy? |
| **Rescatistas** | ¿Qué equipos de rescate operan en Malargüe? ¿Cómo se comunican actualmente? ¿Qué información necesitarían de un sistema como Red Condor? |
| **Defensa Civil** | ¿Qué hace Defensa Civil en Malargüe? ¿Cómo reciben alertas hoy? ¿Podrían integrar un sistema como Red Condor a sus protocolos? |
| **Necesidades reales** | ¿Cuál es el problema concreto que Red Condor debe resolver? ¿Qué funcionalidades son indispensables y cuáles son "deseables"? ¿Cómo validamos que lo que construimos sirve? |

## Actividades

1. Investigar cada tema con fuentes confiables y entrevistas si es posible
2. Elaborar un resumen en `investigacion.md`
3. Identificar dudas en `preguntas-abiertas.md`
4. Proponer aplicaciones para Red Condor en `propuestas.md`
5. Compartir hallazgos con el resto del curso
