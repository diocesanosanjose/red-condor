# Equipo 4: Usuarios

**Objetivo:** Comprender quienes utilizarian el sistema y que problemas concretos necesitan resolver en su contexto real.

## Temas a investigar

| Tema | Que hay que responder |
|------|----------------------|
| **Puesteros** | ?Quienes son los puesteros de Malargue? ?Donde viven? ?Cual es su ciclo anual de trabajo (veranada/invernada)? ?Cuantos dias pasan en la montana sin bajar? ?Que herramientas de comunicacion usan hoy? |
| **Rescatistas** | ?Que equipos de rescate operan en Malargue? ?Como se comunican actualmente? ?Que informacion necesitarian de un sistema como Red Condor? |
| **Defensa Civil** | ?Que hace Defensa Civil en Malargue? ?Como reciben alertas hoy? ?Podrian integrar un sistema como Red Condor a sus protocolos? |
| **Necesidades reales** | ?Cual es el problema concreto que Red Condor debe resolver? ?Que funcionalidades son indispensables y cuales son "deseables"? ?Como validamos que lo que construimos sirve? |

## Actividades

1. Investigar cada tema con fuentes confiables y entrevistas si es posible
2. Elaborar un resumen en `investigacion.md`
3. Identificar dudas en `preguntas-abiertas.md`
4. Proponer aplicaciones para Red Condor en `propuestas.md`
5. Compartir hallazgos con el resto del curso
